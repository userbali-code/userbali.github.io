# Sistem Form Order Terhubung Google Spreadsheet + WhatsApp Sistem ini memiliki fitur: * Input order customer * Data otomatis masuk ke Google Spreadsheet * Tombol kirim ke WhatsApp admin * Rekap order otomatis * Bisa dipakai di HP & laptop --- # 1. Buat Google Spreadsheet 1. Buka Google Spreadsheet 2. Buat spreadsheet baru 3. Buat header seperti ini: | Tanggal | Nama | WhatsApp | Produk | Jumlah | Alamat | Catatan | | ------- | ---- | -------- | ------ | ------ | ------ | ------- | --- # 2. Buka Google Apps Script Klik: Extensions → Apps Script Hapus semua kode lalu ganti dengan kode berikut: ```javascript function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Sheet1"); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date(), data.nama, data.whatsapp, data.produk, data.jumlah, data.alamat, data.catatan ]); return ContentService .createTextOutput(JSON.stringify({ status: "success" })) .setMimeType(ContentService.MimeType.JSON); } ``` --- # 3. Deploy Apps Script Klik: Deploy → New Deployment Pilih: * Type: Web App * Execute As: Me * Who Has Access: Anyone Klik Deploy. Copy URL Web App. Contoh: ```text https://script.google.com/macros/s/AKfycbxxxxxxxx/exec ``` --- # 4. Coding HTML Form Order Ganti: ```javascript const scriptURL = "PASTE_URL_APPS_SCRIPT"; ``` menjadi URL Apps Script milik Anda. --- ```html Form Order
